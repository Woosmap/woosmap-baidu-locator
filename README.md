# Baidu Store Locator

Thanks to Woosmap Javascript SDK : [https://developers.woosmap.com/js-api/get-started.html](https://developers.woosmap.com/js-api/get-started.html)

## Dev Install
### Install dependencies

```ShellSession
$ npm install
```

### Run
```ShellSession
$ npm start
```
Navigate to http://127.0.0.1:8080/

## Production build
```ShellSession
$ npm run build
```
